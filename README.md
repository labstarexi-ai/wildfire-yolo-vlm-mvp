# Wildfire YOLO + VLM MVP

AI-based wildfire detection system using YOLOv8 and Vision-Language Model verification.

---

## Project Structure

- src/        : detection and pipeline logic  
- data/       : images and labels  
- models/     : model weights  
- configs/    : configuration files  

---

## Setup

Create virtual environment:

```bash
python -m venv .venv
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Run

```bash
python src/detect.py
```
